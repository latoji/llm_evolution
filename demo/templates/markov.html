    How this works

    This page uses a character-level n-gram model (orders 1–4) built from two balanced sources: ~150 Project Gutenberg books (classic literature and philosophy) and ~200 Wikipedia articles (science, history, geography, arts, and more). Having both sources gives the model a richer vocabulary — literary prose from Gutenberg, factual and scientific writing from Wikipedia.

    For your input th, the model finds every matching pattern, counts what followed, and turns the counts into probabilities. If the exact context isn’t found it backs off to shorter contexts. The Bigram Transition Matrix shows how the top characters connect: each row is a source character and each cell’s brightness is the probability that the column character follows. The entropy badge measures how spread out the distribution is — near zero means very sure, near 4–5 bits means many options are equally likely. Each Monte Carlo walker independently rolls the weighted die at every step, producing a different path. The convergence chart (shown when walkers finish) tallies which first character most walkers chose.
